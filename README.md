# vinnik.js
This libary was inspired by Oleg Vinnik concert in odessa.

In order to identify is he cool, we created this library.

# Install
```
npm install vinnik.js --save
```

# Usage

```
const vinnik = require('vinnik');

vinnik.isVinnikCool(); 

vinnik.getVinnikImage();
```
